# LINKBRARY 프로젝트

- 관심있어하는 웹들을 폴더별로 정리하여서 관리, 공유 하는 사이트

# 사용스택

### FRONT_END

- NEXT.JS, REACT-QUERY

# 기능구현

- 회원가입, 로그인
- 폴더 추가, 삭제, 이름 변경, 공유
- 링크 추가, 검색
